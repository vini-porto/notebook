What command compiles a Java source file named Main.java?	javac Main.java
What command runs a compiled Java class named Main?	java Main
What keyword in modern Java triggers type inference for local variables?	var
What is the name of the intermediate format produced by javac?	bytecode
What acronym refers to the engine that runs Java bytecode?	JVM
What acronym refers to the full toolkit needed to develop and run Java programs?	JDK
What acronym describes JShell's interactive mode of execution?	REPL
What Java primitive type stores whole numbers?	int
What Java primitive type stores true/false values?	boolean
What Java primitive type stores a single character?	char
What Java primitive type stores decimal numbers?	double
What symbol ends every statement in Java?	;
What type of error is caused by calling a method on a null reference at runtime?	NullPointerException
What Java class represents a resizable list of objects?	ArrayList
What comment syntax is used for a multi-line comment in Java?	/* */
